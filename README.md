# photo-gallery-react

## Demo
[Demo Unsplash API](https://photoflix.netlify.com) (50 requests per hour)

[Demo with mock data](https://photoflix-mock.netlify.com)

## Installation
```bash
npm i
```

## Run dev server
```bash
npm run start
```

## Run build
```bash
npm run build
```

## Run tests
```bash
npm run test
```

## Mock data
On the branch **mock** is enabled the data mocking instead of Unsplash API
